# Quiz Papers

Drop HSC and trial papers here. Accepted formats: PDF, PNG, JPG.

## Naming convention

```
YYYY-[school|hsc]-business.pdf
```

Examples:
- `2024-hsc-business.pdf`
- `2023-hsc-business.pdf`
- `2024-knox-trial-business.pdf`
- `2024-james-ruse-trial-business.pdf`

## What happens next

In the next session, questions from these papers will be parsed and added to the `QUIZ_BANK` array in `src/components/sections/Practice.tsx`. MCQ, short-answer, and extended-response questions each get their own entry with marking criteria pulled from the marking guidelines (drop those here too).
